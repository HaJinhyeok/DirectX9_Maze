## 2026-07-05 - 소스 인코딩과 저위험 오류 정리

### 작업

- CP949 C++ 소스 파일을 UTF-8 BOM으로 변환하고 한글 주석을 복구했다.
- 총알 이동의 중복 호출을 제거했다.
- 만료 총알 제거를 현재 iterator 기준 `erase`로 수정했다.
- 총알 렌더링 반복문의 signed/unsigned 비교를 정리했다.
- X 파일 리소스 배열 해제를 `delete[]`로 수정하고 텍스처 null 검사를 추가했다.
- 미사용 `CalculateDivisionPoints()`의 선형 보간식을 수정했다.

### 검증

- `Debug|x86` 빌드에 성공했다.
- `CalculateDivisionPoints()`는 선언과 정의 외 호출처가 없어 현재 게임 동작에는 영향이 없다.

### 남은 위험

- ANSI DirectX 및 Win32 API를 통한 한글 출력은 실행 환경 확인이 필요했다.
- 플레이어 충돌 처리의 맵 경계 인덱스는 별도 점검이 필요하다.
