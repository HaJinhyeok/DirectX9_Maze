# Roadmap

## 상태 규칙

- `예정`: 아직 시작하지 않음
- `진행 중`: 현재 작업 중이며 동시에 하나만 허용
- `완료`: 완료 조건과 검증을 모두 충족함
- `보류`: 선행 작업이나 추가 판단이 필요함

작업 결과는 `docs/WORKLOG.md`에 기록하고, 완료 조건을 충족했을 때만 상태를 변경한다.
